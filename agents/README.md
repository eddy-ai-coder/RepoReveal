# AI Agents for Documentation Generation

This directory contains AI agent configurations and prompt templates used to generate comprehensive documentation for repositories.

## Files

- `doc-generator-agent.md` - Main agent configuration for documentation generation
- `prompts/` - Directory containing various prompt templates for different documentation sections

## Usage

The agent configurations and prompts are designed to be used with AI coding assistants (like GitHub Copilot, Claude, etc.) to automatically generate comprehensive documentation for well-known repositories.

Each prompt template is structured to extract specific information:
- Repository overview and purpose
- Architecture and design patterns
- Technical implementation details
- Usage guides and examples
- Missing documentation from the original repository
